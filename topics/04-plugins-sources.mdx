# Source Plugins

## Позволяют использовать данные из разных источников через один интерфейс

---

# gatsby-source-filesystem

---
export default FullScreenCode

```json
[
  {
    "name": "Mikhail Novikov",
    "email": "freiksenet@gmail.com",
    "avatar": "pictures/freiksenet.jpg"
  },
  // ...
]
```

---
export default FullScreenCode

```graphql
{
  authorsJson(name: { eq: "Mikhail Novikov" }) {
    name
    email
    avatar {
      publicUrl
    }
  }
}
```

# Transformer Plugins

## Изменяют и дополняют данные из других источников

---

# gatsby-transformer-sharp

---

export default FullScreenCode


```graphql
{
  authorsJson(name: { eq: "Mikhail Novikov" }) {
    name
    email
    avatar {
      childImageSharp {
        fluid(maxWidth: 200) {
          tracedSVG
          srcSet
        }
      }
    }
  }
}
```

---

# gatsby-transformer-remark

---

export default FullScreenCode

```md
\-\-\-
title: "PiterJS встречает Гатсби"
date: 2019-02-19
author: Mikhail Novikov
\-\-\-
Текст
```

---

export default FullScreenCode

```graphql
{
  allMarkdownRemark {
    edges {
      node {
        html
        headings {
          depth
          value
        }
        frontmatter {
          title
          date
        }
      }
    }
  }
}
```
